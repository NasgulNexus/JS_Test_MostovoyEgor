https://nasgulnexus.github.io/JS_Test_MostovoyEgor/

### Установить зависимости

```
    yarn
```

### Для запуска проекта локально необходимо создать файл .env.local

## Записать внутри файла

```
    API_HOST = http://localhost:3000/api
```

### Запустить проект

```
    yarn dev
```
